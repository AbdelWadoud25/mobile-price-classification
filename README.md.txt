# Mobile Price Classification

This project uses Logistic Regression to predict the price range of mobile phones based on various hardware features.

## Dataset
- Source: Kaggle Mobile Price Dataset
- Features: battery_power, ram, px_height, px_width, etc.

## Model
- Algorithm: Logistic Regression
- Accuracy: 0.965

## How to Run
```bash
python mobile_price_model.py
